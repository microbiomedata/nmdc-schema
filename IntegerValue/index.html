Class: integer value

A value that is an integer

URI: nmdc:IntegerValue

Parents

  • is_a: AttributeValue - The value for any value of a attribute for a sample. This object can hold both the un-normalized atomic value and the structured value

Attributes

Own

  • has numeric value 0..1
    • Description: Links a quantity value to a number
    • Range: Float

Inherited from attribute value:

  • has raw value 0..1
    • Description: The value that was specified for an annotation in raw form, i.e. a string. E.g. "2 cm" or "2-4 cm"
    • Range: String
  • was generated by 0..1
    • Range: Activity
  • attribute value➞type 0..1
    • Description: An optional string that specified the type of object.
    • Range: String
    • Example: nmdc:Biosample None
