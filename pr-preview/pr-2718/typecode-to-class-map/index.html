    Typecode to class map

    Schema class definitions include structured patterns that constrain the format of their id values. One element of the structured pattern is the typecode. The following table—which was derived from the schema—shows which schema class can have a given string in the typecode portion of its id values, and which collections can contain instances of that schema class.

    Typecode(s) Schema class Collection name(s)
    bsm Biosample biosample_set
    calib CalibrationInformation calibration_set
    chcpr ChemicalConversionProcess material_processing_set
    chrcon ChromatographyConfiguration configuration_set
    clsite CollectingBiosamplesFromSite collecting_biosamples_from_site_set
    cspro ChromatographicSeparationProcess material_processing_set
    dgms, omprc MassSpectrometry data_generation_set
    dgns, omprc NucleotideSequencing data_generation_set
    dispro DissolvingProcess material_processing_set
    dobj DataObject data_object_set
    extrp Extraction material_processing_set
    filtpr FiltrationProcess material_processing_set
    frsite FieldResearchSite field_research_site_set
    inst Instrument instrument_set
    libprp LibraryPreparation material_processing_set
    manif Manifest manifest_set
    mixpro MixingProcess material_processing_set
    mscon MassSpectrometryConfiguration configuration_set
    poolp Pooling material_processing_set
    procsm ProcessedSample processed_sample_set
    storpr StorageProcess storage_process_set
    sty Study study_set
    subspr SubSamplingProcess material_processing_set
    wfmag MagsAnalysis workflow_execution_set
    wfmb MetabolomicsAnalysis workflow_execution_set
    wfmgan MetagenomeAnnotation workflow_execution_set

    wfmp MetaproteomicsAnalysis workflow_execution_set
