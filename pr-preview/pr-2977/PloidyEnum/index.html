    Enum: PloidyEnum

    Ploidy of an organism's genome. Permissible values map to PATO classes. Used as the range of the ploidy slot on Organism, narrowing the MIxS-imported ploidy (MIXS:0000021) from a free-text/structured pattern to an enumerated set with referential integrity to PATO.
